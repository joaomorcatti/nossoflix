# NossoFlix

Uma página inspirada no layout da Netflix, criada para comemorar 1 ano de namoro. 
Construída do zero com HTML, CSS e JavaScript puro, sem frameworks.

## Tecnologias
  - HTML5
  - CSS3
  - JavaScript

## Estrutura de pastas
```
nossoflix/
├── assets/
│   ├── fonts/
│   ├── images/
│   │   ├── flores/
│   │   ├── numbers/
│   │   └── profiles/
│   ├── scripts/
│   └── styles/
├── .gitignore
├── login.html
└── main.html
```

## O que aprendi

  - Estruturar layouts com Flexbox — usado tanto na página de login quanto no header e carrosséis
  - Usar CSS Grid para organizar o layout geral da página
  - Posicionamento com position: fixed e absolute — header fixo e informações sobre o poster
  - Criar variáveis CSS com :root para manter cores, fontes e espaçamentos consistentes
  - Usar clamp() para tipografia responsiva sem media queries
  - Carregar fontes customizadas com @font-face
  - Manipular o DOM com JavaScript — criar cards dinamicamente a partir de um array de dados
  - Trabalhar com eventos — scroll, keydown e click para interações do modal
  - Usar DOMContentLoaded para garantir que o JS só roda após o HTML estar carregado
  - Criar um modal com abertura, fechamento por botão, Escape e clique fora
  - Travar o scroll do body com overflow: hidden ao abrir o modal
  - Implementar navegação por âncoras com scroll suave
  - Construir um carrossel com scroll horizontal e botões de navegação
  - Organizar um projeto real do zero com HTML, CSS e JS puros — com múltiplas páginas, pastas e assets
